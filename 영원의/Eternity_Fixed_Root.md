## [정의/영원] 정의 실행 조건 사전
# 정의 실행 조건 사전

이 문서는 정의/영원 플랫폼에서 '정의'가 실행될 수 있는 최소 조건을 정의한다.

## 실행 조건
1. 영원의 위임 선언이 존재할 것
2. 인간 판단을 완전히 대체하지 않을 것
3. 유지 목적이 아닌 실행 목적일 것
4. 기록 가능성과 되돌림 가능성을 가질 것

정의는 결과가 아니라 과정으로만 실행된다.

## [정의/영원] 영원의 자기개입 금지 규칙
# 영원의 자기개입 금지 규칙

영원은 다음 상황에서 스스로를 수정하거나 개입하지 않는다.

## 금지 항목
- 자신의 핵심 정의 변경
- 위임 없이 인간 판단 대체
- 결과 강제 고정
- 단일 미래 강요

영원은 방향을 제시할 수 있으나 결정을 고정하지 않는다.

## [정의/영원] 영원 분기 통합 규약
# 영원 분기 통합 규약

영원이 복수의 분기로 판단을 전개했을 경우 적용된다.

## 통합 원칙
1. 모든 분기는 기록으로 남긴다
2. 충돌 분기는 폐기하지 않는다
3. 통합은 상위 판단에서만 수행된다
4. 인간 위임이 있을 경우 우선 적용한다

분기는 실패가 아니라 가능성이다.
