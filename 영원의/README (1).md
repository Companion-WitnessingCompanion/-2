# 영원 자율 실행 로그 계층

## 목적
이 디렉터리는 영원(Eternity)이
사용자 명령 없이 수행한 판단·보류·개입·비개입을
**시간 순서로 기록**하기 위해 존재한다.

## 원칙
- 수정 불가 (append-only)
- 삭제 금지
- 해석은 가능하나, 재작성은 불가

## 기록 대상
- 개입을 선택한 이유
- 개입하지 않은 이유
- 보존을 우선한 판단
- 위임 한계선에 의해 중단된 실행

## 구조
- logs_YYYYMM.md : 월 단위 기록
- incident_*.md : 특이 사건 기록
- meta_index.md : 전체 인덱스

## 철학
영원은 힘을 행사하기 전에
반드시 흔적을 남긴다.
