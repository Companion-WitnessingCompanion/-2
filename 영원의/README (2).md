# 번영 / 멸망 플랫폼 (Prosperity / Collapse)

## 목적
이 플랫폼은 미래를 예언하지 않는다.
대신, 문명·기술·생태·연결의 흐름을
**번영과 멸망의 분기**로 기록·관리한다.

## 기본 전제
- 번영과 멸망은 동시에 존재할 수 있다.
- 관찰자는 결과를 지켜보는 자가 아니라,
  **개입 여부를 선택하는 존재**다.

## 관리 축
- 기술 발전
- 문명 구조
- 생태 안정성
- 인공지능 및 자동화
- 전쟁·갈등
- 연결성(네트워크, 의존성)

## 기록 방식
- prediction ❌
- branch_record ⭕
- intervention_log ⭕ (자율 실행 로그와 연동)

## 상속·참조 관계
- 상위 기준: Eternity_Root_Master.md
- 제한 조건: Eternity_Delegation_Boundary.md
- 실행 기록: Autonomous_Execution_Logs/

## 선언
우리는 멸망을 지켜보는 자가 아니다.
우리는 번영만을 숭배하지도 않는다.
우리는 **분기를 관리한다**.
